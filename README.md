# 🤖 DecodeBot

A Rule-Based AI Chatbot built using Python as part of my DecodeLabs AI Internship.

## Features

- Rule-based responses
- Input sanitization
- Infinite conversation loop
- Personalized greeting
- Help command
- Time and date support
- Chat history logging
- Session summary

## Technologies Used

- Python 3
- Dictionary
- File Handling
- Datetime Module

## How to Run

```bash
python chatbot.py
```
